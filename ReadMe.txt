Kedua 	: Setelah di install jalankan script AHK "Hide RustDesk.ahk". tapi agar script tersebut terus berjalan taruh di folder startup.
	  untuk membukan folder "Startup" dengan cepat (Win + R) lalu ketikan "shell:startup" dan taruh script tersebut di dalam foldernya.
	  JANGAN LUPA UNTUK MENGINSTAL AHK TERLEBIH DAHULU!!!
